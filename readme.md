## Bonsai Pricing Page
A clean and responsive pricing page inspired by the “Bonsai Pricing” design. Built using HTML and CSS, with a focus on layout, typography, and responsive UI design.

## Overview
This project recreates a modern pricing page layout featuring multiple pricing tiers, a clean design, and mobile-friendly responsiveness. It was built using plain HTML and CSS, making it simple, fast, and beginner-friendly.

## Features

Fully responsive pricing layout
Semantic HTML structure
Organized CSS using Flexbox and Grid
Simple and minimal codebase
Works on all devices

## Tech Stack
HTML5
CSS3
